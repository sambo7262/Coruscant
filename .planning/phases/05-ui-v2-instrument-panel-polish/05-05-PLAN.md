**Step 4: Audit NowPlayingBanner**

Confirm collapsed rail is 40px (done in Plan 04). No additional changes needed.

**Step 5: Add CSS overflow protection**

In `index.css`, add:

```css
/* Prevent body scroll on kiosk viewport (D-01) */
body {
  overflow: hidden;
  height: 100vh;
  max-height: 480px; /* kiosk target — overridden by larger viewports naturally */
}

@media (min-height: 481px) {
  body {
    max-height: none;
    overflow: auto;
  }
}
```

This ensures at exactly 480px viewport height, the body cannot scroll. On larger viewports (desktop, phones in portrait), normal scrolling is allowed.
