**WiringOverlay.tsx:**

  - `position: fixed`, `inset: 0`, `z-index: 0` (same layer as background)
  - `pointer-events: none`

  - `width="100%"`, `height="100%"`

- Paths should be geometric: primarily horizontal and vertical lines with rounded corner curves (use SVG `M`, `L`, `Q` or `C` commands)

- Some paths should be thicker bundles: `strokeWidth: 2.5` with `stroke: rgba(232, 160, 32, 0.05)`
- Paths should loosely connect panel areas — e.g., run from upper-left toward center, from lower-right across bottom, etc.
- Use viewBox="0 0 800 480" to match the primary 800x480 viewport
- Paths are STATIC — no animation, no interaction
